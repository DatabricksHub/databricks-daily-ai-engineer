# README.md: Real-Time Analytics Dashboard using Databricks Lakehouse
## Overview
This demo project showcases the capabilities of Databricks Lakehouse in building a real-time analytics dashboard for sales data. The project combines the best of data warehouses and data lakes to provide a unified platform for data, analytics, and AI.

## Why this feature matters
The Lakehouse feature matters because it enables organizations to:
* Unify their data and analytics platforms, reducing complexity and costs
* Provide real-time insights and analytics, enabling data-driven decision-making
* Support a wide range of use cases, from data warehousing to data science and machine learning

## Prerequisites
To run this demo project, you will need:
* A Databricks account with access to a Lakehouse-enabled workspace
* A basic understanding of Databricks notebooks and SQL
* A sample sales data dataset (provided in the project repository)

## How to run
To run this demo project, follow these steps:
1. **Import the notebook**: Import the `lakehouse_dashboard_notebook` into your Databricks workspace.
2. **Create a sample dataset**: Run the `create_sample_data` SQL query to create a sample sales data dataset.
3. **Run the notebook**: Run the `lakehouse_dashboard_notebook` to create the real-time analytics dashboard.
4. **Execute SQL queries**: Execute the provided SQL queries to analyze the sales data and generate insights.

## Expected output
The expected output of this demo project is a real-time analytics dashboard that displays:
* Sales data summarized by region, product, and time period
* Real-time updates to the dashboard as new sales data is ingested
* Interactive visualizations and filters to enable exploratory data analysis
