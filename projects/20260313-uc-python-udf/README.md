# README.md: UC Python UDF Demo Project
=====================================

## Overview
This demo project showcases the use of User-Defined Functions (UDFs) in Python with Databricks Unity Catalog (UC). The project builds a data pipeline that utilizes custom data validation using UC Python UDFs.

## Why this feature matters
The ability to create custom User-Defined Functions in Python is crucial for extending the functionality of Databricks Unity Catalog. By leveraging UC Python UDFs, data engineers and analysts can:

* Simplify complex data transformations and validation logic
* Improve data quality and consistency across the organization
* Enhance the overall efficiency of data pipelines and workflows

## Prerequisites
To run this demo project, you will need:

* A Databricks workspace with Unity Catalog enabled
* A cluster with Python and SQL support
* The necessary permissions to create and execute notebooks and SQL queries

## How to run
To run this demo project, follow these steps:

1. **Notebook**: Open the provided Python notebook and execute the cells to create the custom UDFs and data pipeline.
2. **SQL**: Execute the SQL queries provided to test the custom UDFs and validate the data pipeline.

## Expected output
After running the notebook and SQL queries, you should see the following output:

* A validated dataset with custom data validation applied using the UC Python UDFs
* A summary of the data pipeline execution, including any errors or warnings
