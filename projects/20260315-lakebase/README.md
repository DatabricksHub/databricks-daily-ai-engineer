# Lakebase Demo Project: Building a Scalable Data Warehouse
==============================================

## Overview
This demo project showcases the capabilities of Lakebase, a cloud-native, open, and collaborative database offering from Databricks. The project demonstrates how to build a scalable data warehouse using Lakebase for real-time analytics and business insights.

## Why this feature matters
Lakebase is a game-changer for data engineering, data science, and analytics teams. It provides a unified platform for data warehousing, data lakes, and data science workloads, enabling real-time analytics and business insights. With Lakebase, teams can:

* Build scalable and secure data warehouses
* Perform real-time analytics and data science workloads
* Collaborate seamlessly across teams and departments
* Take advantage of a cloud-native, open, and extensible architecture

## Prerequisites
To run this demo project, you will need:

* A Databricks account with Lakebase enabled
* A cluster with Lakebase configured
* Basic knowledge of Databricks, SQL, and data warehousing concepts

## How to run
This demo project consists of a notebook and SQL scripts that demonstrate how to build a scalable data warehouse using Lakebase.

### Notebook
1. Clone this repository and upload the notebook to your Databricks workspace.
2. Attach the notebook to a cluster with Lakebase configured.
3. Run the notebook to create a sample dataset and perform data warehousing tasks.

### SQL
1. Open a new SQL query window in your Databricks workspace.
2. Run the provided SQL scripts to create tables, load data, and perform analytics queries.

## Expected output
The demo project will create a sample dataset and perform various data warehousing tasks, including:

* Creating tables and loading data
* Performing analytics queries and aggregations
* Demonstrating real-time analytics and business insights

The expected output will include:

* A sample dataset with various tables and data
* Query results and analytics insights
* Demonstrated scalability and performance of Lakebase
