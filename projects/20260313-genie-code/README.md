# Automating Data Pipeline Creation using Genie Code
## Overview
This demo project showcases the capabilities of Genie Code, a new feature in Databricks that enables users to generate high-quality code for data engineering and machine learning tasks using a simple and intuitive interface. The project demonstrates how to automate the creation of a data pipeline for a sample e-commerce dataset using Genie Code.

## Why this feature matters
Genie Code matters because it simplifies the process of creating data pipelines and machine learning models, making it more accessible to users who are not experienced in data engineering or machine learning. By automating the code generation process, Genie Code saves time and reduces the risk of errors, allowing users to focus on higher-level tasks such as data analysis and insights generation.

## Prerequisites
To run this demo project, you will need:
* A Databricks account with access to Genie Code
* A cluster with the necessary dependencies installed (e.g. Apache Spark, MLlib)
* The sample e-commerce dataset (provided in the project repository)

## How to run
To run this demo project, follow these steps:
1. **Notebook**: Open the `genie_code_demo` notebook in your Databricks workspace and run each cell in sequence. The notebook will guide you through the process of creating a data pipeline using Genie Code.
2. **SQL**: Alternatively, you can run the SQL queries provided in the `genie_code_demo.sql` file to create the data pipeline.

## Expected output
The expected output of this demo project is a fully functional data pipeline that ingests, processes, and analyzes the sample e-commerce dataset. The pipeline will generate insights and visualizations that can be used to inform business decisions.
